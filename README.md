# 🧠 NeuroLearn AI

An AI-powered smart learning assistant that converts study notes into interactive exams.

## 🚀 Features
- AI-generated MCQs (Groq LLM)
- Google Forms-style exam mode
- Timer-based tests
- Leaderboard system
- Weak topic detection

## 🛠 Tech Stack
- Python
- Streamlit
- Groq API
- Pandas

## 🎯 Goal
Make studying interactive, adaptive, and fun.

## 🔥 Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
